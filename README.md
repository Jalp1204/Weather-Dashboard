# 🌤️ Weather Dashboard

A modern weather dashboard built using **HTML, CSS, and JavaScript** that fetches real-time weather data from the OpenWeather API. Users can search for any city or use their current location to view weather conditions along with a 5-day forecast.

## ✨ Features

- 🔍 Search weather by city name
- 📍 Get weather using current location
- 🌡️ Current temperature
- 🤒 Feels like temperature
- 💧 Humidity
- 🌬️ Wind speed
- 👀 Visibility
- ☁️ Weather description and icon
- 📅 Current date
- 📆 5-Day Weather Forecast
- ⌨️ Press Enter to search
- ⏳ Loading indicator
- ⚠️ Error handling for invalid cities and location permission

## 🛠️ Built With

## ⚙️ Setup

1. Clone the repository

```bash
git clone https://github.com/Jalp1204/weather-dashboard.git
```

2. Navigate to the project folder

```bash
cd weather-dashboard
```

3. Create a `config.js` file

```javascript
const API_KEY = "YOUR_OPENWEATHER_API_KEY";
```

4. Open `index.html` in your browser.
